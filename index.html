Tentang

Hai!! Perkenalkan nama saya Ainnuna Elya Khansa. Saya adalah mahasiswa program pendidikan S1 Informatika angkatan 2021 di Universitas AMIKOM Yogyakarta. Lulusan SMK jurusan Rekayasa Perangkat Lunak (RPL) di SMK Muhammadiyah 1 Muntilan.

Saat SMK dahulu saya dikirim PKL (Praktek Kerja Lapangan) oleh guru ke perusahaan AXIOO yang berada di Graha Mabito, Bandung, Jawa Barat, pada bulan September-Desember, tahun 2019. Saya di Graha Mabito ditempatkan di divisi IT bersama dengan 3 teman sekolah saya yang lainnya.

Saat kuliah, pada tahun pertama kuliah di Semester 1-2 saya mengikuti organisasi UKM (Unit Kegiatan Mahasiswa) AMCC (AMIKOM Computer Club) di divisi Hardware-Software. Pada Semester 1, saya menjadi sekretaris panitia event yang diadakan oleh AMCC yaitu AGS 2022 dan pada Semester berikutnya atau Semester 2 saya kembali menjadi sekretaris panitia event yang diadakan juga oleh AMCC yaitu CODE 2022. Pada Semester 3, saya juga mengikuti kegiatan menjadi Asisten Praktikum untuk mata kuliah Algoritma dan Pemrograman pada Semester 1 di Program Studi Informatika. Pada Semester 4, saya kembali menjadi Asisten Praktikum di mata kuliah Komputer Grafis, Pemrograman, dan Struktur Data pada Semester 2 di Program Studi Informatika.
